# Wishlist 2026

En enkel ønskeliste-applikasjon bygget med Spring Boot.

## Om Prosjektet

Dette er en webapplikasjon for å administrere ønskelister. Applikasjonen er bygget med Spring Boot og bruker følgende teknologier:

- Java 21
- Spring Boot 3.5.7
- Spring Data JPA
- H2 Database
- Lombok
- Maven

## Kom i Gang

For å kjøre prosjektet lokalt:

1. Klone prosjektet:
   ```bash
   git clone https://github.com/bitruls/wishlist_2026.git
   ```

2. Naviger til prosjektmappen:
   ```bash
   cd wishlist_2026
   ```

3. Bygg prosjektet med Maven:
   ```bash
   ./mvnw clean install
   ```

4. Kjør applikasjonen:
   ```bash
   ./mvnw spring-boot:run
   ```

Applikasjonen vil starte på `http://localhost:8080`
